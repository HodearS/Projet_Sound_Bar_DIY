Objectif du projet :

Réaliser une bar de son dolby atmos qui fait à la fois home cinéma et enceinte Bluetooth.
Afin de pouvoir contourner les frais de licence liés au dolby atmos, le projet se base sur une architecture Texas Instruments.
Une version custom de la carte d'évaluation (EVMK2GX) du processeur 66AK2Gx sera utilisée.
Il s'agit dans ce projet de la carte processeur.
L'objectif est d'utiliser les programmes de tests du dolby atmos fournis par Ti.
Ce projet correspond à l'intégralité du schéma de la carte d'évaluation. Il servira de base saine pour la suite du projet.

*Le schéma n'est pas destiné à être routé ni produit tel quel.

Source du schéma :
https://www.ti.com/tool/EVMK2GX#design-files

Project objective:

Create a dolby atmos sound bar that doubles as a home theater and Bluetooth speaker.
The project is based on a Texas Instruments architecture, in order to bypass the licensing fees associated with dolby atmos.
A custom version of the evaluation board (EVMK2GX) for the 66AK2Gx processor will be used.
This project specifically concerns the processor.
The aim is to use the dolby atmos test programs supplied by Ti.
This project matches the entire evaluation board schematic. It will serve as a sound basis for the rest of the project.

*The schematic is not intended to be routed or produced as is.

Source of the schematic :
https://www.ti.com/tool/EVMK2GX#design-files
